# mojoPortalBlazor
This is a [mojoPortal](https://github.com/i7MEDIA/mojoportal) inspired remake in Blazor and .net 5.0.  mojoPortal Blazor Edition is a an extensible, cross database, mobile friendly, web content management system (CMS), and web application framework written in C# ASP.NET.
